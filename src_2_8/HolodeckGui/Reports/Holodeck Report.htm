Holodeck Enterprise Edition Report



API Log Summary Table

 This report shows summary data for APIs logged by each process.
Use this data to determine how many API calls were made for each category.
Use the '+' button in the Type column to see the function names that were called in each category.



Faults Summary Table

 This report shows summary data for the faults set for each process.
Use this data to determine which faults were active at the time of report creation.



Tests Summary Table

 This report shows summary data for the tests set for each process.
Use this data to determine which tests were active at the time of report creation.



Error Codes Summary Table

 This report shows summary data for the error codes set by APIs called from each process.
Use this data to examine any API calls that set failure error codes.
Use the '+' button in the Error Code Type column to see what error codes were set.
Then use the '+' button in the Error Code column to see which functions set each error code.



Resource Dependencies Summary Table

 This report summarizes the dependencies for each process
Use this data to see the number of dependencies as well as the number of times each dependency was touched. If you see a dependency with a very high hit-count you may be looking at a performance bug.
Use the '+' button in the Type column to view hit count for each dependency.



Resource Usage Summary Table

 This report shows maximum and average resource usage for each process.
Click the '+' button below each process to view resource usage numbers



Return Values Summary Table

 This report shows summary data for the values returned by APIs called from each process.
Use this data to examine any API calls that returned failure error codes.
Use the '+' button in the Return Value Type column to determine what return values were set.
Then use the '+' button in the Return Value column to see what API calls failed.
