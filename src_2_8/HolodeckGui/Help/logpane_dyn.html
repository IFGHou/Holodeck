Log Pane Help
As soon as you begin the AUT Holodeck will start to log the API calls you have specified to be logged. The log pane will hold any logs you are currently viewing. Logs can be filtered, sorted, or exported to make working with the data Holodeck provides as easy as possible.

Tips
To sort a column, click the column, a grey arrow indicates sort order.

To create a test based on a log entry double click the log item

To Filter a column right-click the column header and select Filter Column.

Log Filters vs. AutoFilters
 Log filters change which API functions Holodeck intercepts. Logging fewer functions will allow the application and Holodeck to run more quickly; Logging more functions will allow you to turn on additional tests and faults.

Autofilters change which log entries are displayed. This is useful for investigating failures by limiting the set of API functions being displayed.
