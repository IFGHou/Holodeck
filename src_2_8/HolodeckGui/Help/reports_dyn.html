Tips
Right Click either the reports node or the process node and select Create Report to create a report at anytime.

Use the right click menu or delete key to delete one report, or all the reports for the project.

If you want to create a new report without overwriting an old report, first rename the old report, so it is not overwritten

Reports span an entire project for easy comparison of multiple threads and processes
