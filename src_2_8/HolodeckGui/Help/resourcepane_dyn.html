Resource Help
Holodeck can also track external file, library, and registry resources and log every read, write, update, or access the application makes. The resource pane shows al these resources in an easy to read tree view.

Tips
Jump to the last log entry that touched a resource by right clicking the resource in the resource pane and selecting “Select most recent log entry”

You can set a resource fault easily by finding the resource in the treeview right clicking and selecting the desired fault.

Check which files and Registry entries your program touches, reads, writes, or modifies in the Resource Pane.

Look in the property pane to see how many times a resource was touched and what the last API call was that used this resource.

Resources will not be tracked if logging for these categories is turned off.
