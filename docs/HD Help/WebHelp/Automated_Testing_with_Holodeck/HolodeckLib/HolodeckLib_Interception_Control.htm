HolodeckLib Interception Control

bool InterceptFunction (String * functionName, String * replacementFunctionName, String * replacementDllName);

·      Description - Intercepts a function and replaces with the replacement function

o     functionName – the name of the function you are replacing

o     replacementFunctionName – the name of the replacement function

o     replacementDllName – the name of the dll in which the replacement function resides.

·      Return Value - Returns true if all operations are successful, false otherwise
