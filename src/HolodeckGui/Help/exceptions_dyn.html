Exceptions Help
Holodeck catches all second chance exceptions your application produces, and can be configured to catch all exceptions. If Holodeck intercepts first chance exceptions the exceptions are routed through Holodeck first, as a first chance exception, then passed back to the application to handle. If the application does not handle the exception it is logged as a second chance exception. Otherwise Holodeck will only log second chance exceptions after the application has not handled them.
