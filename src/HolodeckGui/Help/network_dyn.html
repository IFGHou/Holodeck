Network Corruption Help
Holodeck can set faults on the data being transmitted over the network. Holodeck does this by corrupting the actual data being sent and received on all, or specific ports.

Tips
To delete a network corruption test Right click the Network Corruption Fault you would like to delete and select Delete Network Corruption Fault.

Use Network Corruption to find bugs in corrupted packet handling code. Try using different levels of corruption to find bugs deeper in the code base.
