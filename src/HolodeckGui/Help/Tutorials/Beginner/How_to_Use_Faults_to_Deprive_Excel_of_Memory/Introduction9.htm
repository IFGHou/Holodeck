Introduction

This tutorial will cover how to deprive an application of memory by setting the Insufficient Memory fault. The Insufficient Memory fault fails any calls to API functions that try to allocate additional physical memory or virtual memory. Holodeck will not fail functions the de-allocate memory or re-allocate the same amount of memory.

 

To start this project create a new project using Microsoft Excel and default logging. Excel’s default location is "C:\Program Files\Microsoft Office\Office10\EXCEL.EXE."

 

For more information on how to create a project please see the Setting up your first project tutorial.
