Corrupted Files Help
When you create a File corruption fault Holodeck does not corrupt the original file, but rather when the Application Under Test attempts to access that file it redirects the Application Under Test to a corrupted version of the file. The Corrupted Files Pane shows all the corrupted files Holodeck has created for the application under test.

Tips
To create a file corruption fault click Application > Create a New File Corruption Fault… This will bring up the file corruption wizard.

After creating a File Corruption Fault you can access the resulting corrupted file by expanding the Corrupted Files node in the Project pane.
