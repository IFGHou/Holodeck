Test Pane Help
Tests are a way to inject targeted failures; a test targets a single API call based on matching the in parameters you specify. If it matches the result, it will output the return value, error codes, and out parameters you specify.

Tips
Delete a test quickly by highlighting the test and pressing the delete key.

Tests can be modified after creation by right clicking the test and selecting Modify Test or by double clicking the test entry.

Native api tests can cause a failure by modifying out parameters, return values, and error codes.

.net api tests can cause a failure by modifying out parameters, return values, and throwing exceptions.
